# ABC BOOTCAMP

ABC 부트캠프 데이터 탐험가 과정 3기에서 수행한 코드 모음입니다.  
기온 데이터, 인구 구조, 교통사고, 뉴스/댓글 크롤링, 음악 차트 분석 등을 실습하며 데이터 처리 파이프라인을 정리했습니다.

---

## 학습 주제

- 파이썬 기초 문법 및 제어문  
- 데이터 수집(크롤링) 및 전처리  
- pandas / numpy 기반 데이터 분석  
- matplotlib / seaborn 시각화  
- 공공데이터 기반 분석(기온, 인구, 교통사고 등)  
- 텍스트 데이터 분석(뉴스/댓글)  
- 미니 프로젝트 기반 문제 해결
